---
title: 'Blocktyp: Berechnung'
---

### Einführung

Der Blocktyp "Berechnung" ist ebenfalls ein Mitglied der Kategorie "Daten", denn auch er generiert Variablen. Der Unterschied hier ist, dass du Zahlen und andere Variablen in einem mathematischen Ausdruck kombinieren kannst, um eine neue Variable zu schaffen.

### Unterstützte Grundrechenarten

Der Block bietet Unterstützung für die folgenden mathematischen Operationen:

* \+ für Addition
* \- für Subtraktion
* \* für Multiplikation
* / für Division
* () für Klammern zur Steuerung der Reihenfolge der Operationen

### Berechnung erstellen

1. Berechnungsblock erstellen: Füge einen neuen Block des Typs "Berechnung" hinzu.
2. Formel einfügen: Im Formel-Editor gibst du die mathematische Formel ein, die Zahlen und/oder Variablen sowie Rechenoperationen enthält.
3. Variable Referenzieren: Die generierte Variable kann in Verbindungsbedingungen oder Textelementen referenziert werden.

### Anwendung und Ergebnis

Sobald das Formular den Berechnungsblock erreicht, wird die Formel ausgewertet und das Ergebnis in der entsprechenden Variable gespeichert. Diese Variable kann dann in weiteren Verbindungen oder Texten innerhalb deiner Anwendung verwendet werden.
